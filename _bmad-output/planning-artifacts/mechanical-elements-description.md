# Description des éléments mécaniques et de leurs interactions

## Les grands types d'éléments

### **Edge**

Un élément de type edge est défini par les positions de ses 2 extrémités. On peut placer des nodes à ses extrémités. La poutre, en particulier, a la propriété de permettre de placer des nodes sur toute sa longueur, en plus de ses extrémités.

Placement: Un edge est placée en 2 étapes, une extrémité puis l'autre. Au placement de chaque extrémité, si elle est dans la hit-box d'un node, la connextion se fera automatiquement.

Déplacement: On peut déplacer un edge en l'attrapant sur sa longueur, il gardera son angle et sa longueur. Ou on peut le déplacer par une de ses extrémités, auquel cas l'autre restera fixe. (Dans ce cas la longueur de l'edge pourra changer librement.)

Structure de Données: `id`, `type`, `positionStart`, `positionEnd`

Les éléments de type Edge sont : \[Beam, Spring, Damper, Belt\]

### **Node**

Dans les nodes, sont inclus les éléments définis uniquement par une position (et potentiellement un angle). On peut les placer aux extrémités des edges ou sur la longueur d'une poutre.

Placement: Un node peut être placé à l'extrémité d'un edge ou sur la longueur d'une poutre. Au placement du node, si il est dans la hit-box de l'extrémité d'un edge ou dans la hit-box d'une poutre, la connextion se fera automatiquement. De plus, on peut placer un node sur une intersection entre 2 poutres pour se lier aux 2 en même temps. Un node ne peut pas être placé sur un autre node sauf exception.

Déplacement: On peut déplacer un node en cliquant dessus et en le draggant vers la position voulue. Les éléments liés au node les contraintes de déplacement.

Structure de Données: `id`, `type`, `isGrounded`, `position`

Les éléments de type Node sont : \[Slider, Pivot, Slidep, Join, Gear, Mass\]

### **Ground**

La terre (Ground) n'est pas un élément comme les autres. C'est plutot un paramètre qu'on peut appliquer à aux nodes.
